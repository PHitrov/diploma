Hello World! Привет мир!
Это тестовый файл для проверки работы переводчика и озвучки.

Основные функции для тестирования:
1. Перевод коротких фраз: "Good morning" -> "Доброе утро"
2. Технические термины: "Machine Learning model accuracy is 98.5%"
3. Вопросы: "What time is it?" -> "Который час?"
4. Идиомы: "Break a leg!" -> "Ни пуха ни пера!"
5. Числа и единицы: "Distance: 10 km" -> "Расстояние: 10 км"
6. Специальные символы: "C++ & Python <3" -> "C++ и Python <3"
7. Многоязычный текст: "I use SQL for базы данных"
8. Научные термины: "Photosynthesis equation: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂"
9. Имена собственные: "Albert Einstein said E=mc²"
10. Восклицания: "Wow! Это удивительно!"
